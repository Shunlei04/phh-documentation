Getting Started with Proxy App Step 1: How to download PHH Proxy App Click here to download PHH Proxy App
After clicking “Download”, the “Recent download history” will appear and click on “Full download history”. After the download had finished, click on “PHH Proxy.exe”. Step 2: How to install PHH Proxy App
After clicking on the exe file, there are two possible outcomes as shown in the above figures. You can click on “Run”. Step 3: Sending Request Token to Admin After running the application, this window will appear and click on “Request Token”. After requesting the token, you will have to wait 30 seconds to be able to abort request. After 30 seconds, you can click on the button if you want to abort the request. Step 4: Accepting Request From Admin Side Now, the admin will have to accept the request from the user in Proxy User Requests. The admin must wait on Proxy Users Requests page before user making token request in order to see incoming request. The information from the user’s device will be showed and click on it. The device information will be showed again and you can accept or reject the request. In the “Proxy Server Users”, you can see the devices that are already accepted. Click on menu icon to show detail information.
In the “Proxy User Groups”, you can add user to the desired group by clicking on menu icon.
Click on the check box of the desired user. Step 5: PHH Proxy App Useage Guide After the admin had accepted the user request and added to the user group, the user will be able to see the available apps. If there is no app showing, click on the refresh icon. These apps will be showed according to the app group added by the admin. You can click on the desired app and the window will redirect to the relative application. For example, if you click on “Facebook”, the Facebook app will be showed. If you want to return to the available apps again click on “Apps”.
You can also click on the new window icon to open the app in new window.
Alternatively, you can also click on “Window” to call new window.
